# Lego sequencer

## files

### format:

* filenames:
	* contain the original sequences (e.g white - white -white -blue -green - ... etc)
	* all sequences start with multiple white blocks to avoid errors at the start of the sequence
	* some sequences were recorded at different speeds (indicated as _speed10, _speed20 etc)
* each .csv file
	* column 1 = time
	* column 2 = recorded colour signal
	* column 3 - ... = colour signals of replicated runs

## colour encoding: 

* 2 = Blue
* 3 = Green
* 4= Yellow
* 5 = Red
